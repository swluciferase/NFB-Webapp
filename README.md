# NFB-Webapp

Neurofeedback Web Application — part of the [SigmaCog](https://www.sigmacog.xyz) platform.

## Integrated Modules

| Module | Repo | Function |
|--------|------|----------|
| VisioMynd | [rppg-webapp](https://github.com/swluciferase/rppg-webapp) | rPPG heart rate & HRV measurement |
| ViboMynd | [BNB-Web](https://github.com/swluciferase/BNB-Web) | Vibration/sound neurofeedback |
| SigmaCog EEG | [sgimacog-web](https://github.com/swluciferase/sgimacog-web) | Brainwave measurement & analysis |
